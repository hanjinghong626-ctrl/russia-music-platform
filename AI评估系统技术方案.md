# AI 评估系统技术方案

## 功能概述

实现双轨评估系统：
1. **AI自动评估（免费）**：音准、节奏、音色分析 + 俄罗斯留学适配建议
2. **人工评估（付费）**：俄罗斯专家点评 + 微信对接

## API 接口

### 1. AI音频分析

**端点**: `POST /api/analyze`

**请求**:
- Content-Type: `multipart/form-data`
- 字段:
  - `audio`: 音频文件 (MP3, WAV, M4A, FLAC)
  - `type`: 分析类型 (默认: `vocal`)

**响应**:
```json
{
  "success": true,
  "analysis": {
    "score": 78,
    "pitch_score": 82,
    "rhythm_score": 75,
    "timbre_score": 78,
    "strengths": ["演唱完整性较好", "音量控制稳定"],
    "weaknesses": ["高音区控制待提升"],
    "suggestions": ["每天进行音阶练习"],
    "russia_fit": "适合申请俄罗斯音乐学院",
    "suitable_schools": ["莫斯科国立柴可夫斯基音乐学院"],
    "audio_info": { "duration": 120, "sample_rate": 44100 }
  }
}
```

### 2. 人工评估申请

**端点**: `POST /api/human-review`

**请求**:
```json
{
  "name": "张三",
  "phone": "13800138000",
  "wechat": "zhangsan123",
  "email": "zhangsan@example.com",
  "audio_level": "intermediate",
  "target_school": "莫斯科柴可夫斯基音乐学院",
  "description": "学习声乐3年..."
}
```

**响应**:
```json
{
  "success": true,
  "message": "申请已提交，我们将在24小时内与您联系",
  "application_id": "APP_1712000000000"
}
```

## 前端功能

### 评估页面 (/evaluate)

**Tab 1: AI自动评估**
- 拖拽/点击上传音频
- 音频预览播放
- 本地特征实时显示
- 后端AI分析
- 结果可视化（评分条、优势/不足、建议）

**Tab 2: 人工评估**
- 申请表单（姓名、手机、微信号必填）
- 表单验证
- 提交状态反馈

## 环境变量

```env
# 扣子AI API (可选，未配置时使用本地分析)
Coze_API_Token=your_token_here
Coze_Bot_ID=your_bot_id_here
```

## 音频特征提取

前端使用 Web Audio API 提取特征：
- **RMS**: 均方根音量
- **Peak**: 峰值音量
- **频谱分布**: 低/中/高频能量比
- **基频估算**: 音高检测

后端基于 WAV 文件头解析：
- 采样率
- 通道数
- 位深度
- 数据长度
